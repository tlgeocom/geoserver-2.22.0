GWC S3 BlobStore plugin
========================
This plugin supports the use of the `AWS <https://aws.amazon.com>`_ Simple Storage Service (`Amazon S3 <https://aws.amazon.com/s3/>`_) as storage medium for :ref:`GWC<gwc_webadmin>`.
